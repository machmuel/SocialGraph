## 3. Findings (Prioritized)

### F1 (High): Graph container semantic role could suppress child interactive semantics for assistive technology

Evidence before fix:

- Graph root SVG was configured as `role="img"` while graph nodes/links are interactive descendants.

User impact:

- screen-reader users may receive a flattened image representation instead of reliable child control semantics for nodes/links.

Status: **Fixed in this heartbeat**

Implementation:

- changed graph root semantic role to `role="group"` to preserve discoverable interactive child elements.

### F2 (Medium): Relationship labels in graph edge rendering could degrade to empty/undefined copy

Evidence before fix:

- edge text used raw `link.label` while interaction labels already used a fallback (`edgeLabel`).

User impact:

- inconsistent graph annotation quality when relationship kind is missing or malformed.

Status: **Fixed in this heartbeat**

Implementation:

- aligned edge text rendering to use the same `edgeLabel` fallback path as ARIA/interaction labels.

## 4. Implemented Changes
